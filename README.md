# 📊 Corporate Compliance Monitoring Dashboard

A Power BI dashboard project designed to monitor, analyze, and track corporate compliance activities across different regulatory frameworks such as Company Law, SEBI LODR, FEMA, GST, Labour Law, RBI Regulations, and TDS.

This dashboard helps organizations monitor filing status, departmental responsibility, penalties, and compliance trends throughout the year.

🎯 Project Objective

The objective of this project is to create a centralized compliance monitoring system that helps management:

- Track total compliances across departments
- Identify pending and overdue filings
- Monitor penalties due to non-compliance
- Analyze compliance distribution by law category
- Improve regulatory risk management

---

📌 Key Features

- ✔ KPI Cards for quick compliance insights  
- ✔ Filing Status Analysis (Completed, Pending, Overdue, Upcoming)  
- ✔ Law Category-wise Compliance Monitoring  
- ✔ Department-wise Compliance Responsibility  
- ✔ Monthly Compliance Tracking  
- ✔ Penalty Analysis by Law Category  
- ✔ Interactive Filters (Law Category, Department, Filing Status, Frequency, Month)

---

📊 Dashboard Metrics

The dashboard tracks the following key metrics:

- Total Compliances  
- Completed Compliances  
- Pending Compliances  
- Overdue Compliances  
- Upcoming Compliances  
- Non-Compliance Penalty Amount

---

📈 Visualizations Included

The dashboard contains the following visual elements:

1️⃣ KPI Summary Cards  
   - Displays overall compliance performance.

2️⃣ Filing Status Distribution  
   - Donut chart showing compliance status: Completed, Pending, Overdue, Upcoming

3️⃣ Compliance by Law Category  
   - Bar chart comparing compliances across:
     - Company Law
     - SEBI LODR
     - FEMA
     - GST
     - Labour Law
     - RBI Regulations
     - TDS

4️⃣ Compliance by Department  
   - Shows responsibility across departments:
     - Finance
     - Compliance
     - HR
     - Secretarial
     - Legal

5️⃣ Monthly Compliance Trend  
   - Tracks compliance activities throughout the year.

6️⃣ Non-Compliance Penalty Analysis  
   - Displays penalty impact by law category.

🧾 Dataset Details

The dataset used in this project contains the following fields:

| Column                 | Description                          |
|------------------------|--------------------------------------|
| Compliance ID          | Unique compliance identifier         |
| Law Category           | Applicable regulation                |
| Designated Department  | Responsible department               |
| Filing Status          | Completed / Pending / Overdue / Upcoming |
| Due Date               | Compliance deadline                  |
| Filing Date            | Date of completion                   |
| Penalty Amount         | Penalty for non-compliance           |
| Frequency              | Monthly / Quarterly / Annual         |
| Month                  | Compliance reporting month           |

🚀 How to Use

1. Download the repository.  
2. Open `Yearly Compliance Tracker.pbix` in Power BI Desktop.  
3. Ensure `Yearly_Compliance_Tracking_Data_2026.xlsx` (or your data source) is in the correct path or update the Power BI data source connection.  
4. Refresh the data to interact with the dashboard.  
5. Publish to Power BI Service and configure a gateway and scheduled refresh if needed.
